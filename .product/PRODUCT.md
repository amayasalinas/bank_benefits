# Eliseo

## Transformation Thesis
**From**: Colombianos bancarizados con múltiples tarjetas de crédito/débito que no saben cuál usar en cada compra. Los beneficios (cashback, descuentos en restaurantes, descuentos en negocios, puntos, acceso a salas VIP) se desperdician porque la información está fragmentada entre las páginas web de cada banco y las plataformas de las franquicias (Visa, Mastercard, Amex, Diners). Nadie revisa 5 sitios antes de pagar. El resultado: dinero perdido en cada transacción.

**To**: Una sola app donde el usuario ve todos sus beneficios consolidados (banco + franquicia) y sabe al instante qué tarjeta usar para maximizar el valor de cada compra. Como MaxRewards en EE.UU., pero diseñado para el mercado colombiano.

## Target User
Colombianos bancarizados entre 25-45 años con 2 o más tarjetas de crédito/débito. Personas que quieren aprovechar lo que ya tienen pero no tienen tiempo ni herramientas para hacerlo. Sienten que están dejando dinero sobre la mesa con cada compra.

## Product Power Score
- **ΔState**: 9/10 — De información completamente fragmentada e inaccesible a recomendación instantánea y consolidada. El delta es masivo porque hoy literalmente no existe una forma práctica de resolver esto en Colombia.
- **Emotional Intensity**: 10/10 — El usuario califica el dolor como existencial. Cada compra mal optimizada es dinero perdido. La frustración se acumula con el tiempo.
- **Problem Frequency**: 10/10 — Constantemente. Cada transacción con tarjeta es un momento de decisión donde el beneficio se aprovecha o se pierde.
- **Score**: 900 — **Exceptional**

## Current Landscape
- **En Colombia**: No existe competencia directa. La única opción es entrar manualmente a la página web de cada banco para buscar beneficios, y luego revisar las páginas de cada franquicia (Visa, Mastercard, etc.). La información es difícil de encontrar incluso dentro de esas páginas.
- **En EE.UU.**: MaxRewards resuelve este problema con éxito, lo que valida la demanda del mercado.
- **Apps bancarias colombianas**: Cada banco tiene su propia app, pero no se comparan entre sí y no incluyen beneficios de franquicia.
- **Resultado**: La gente usa la tarjeta que tiene a la mano, no la que maximiza el beneficio.

## Unique Advantage
1. **Simplicidad radical**: Todo consolidado en un solo lugar — beneficios de banco + franquicia. Sin fricción.
2. **First mover en Colombia**: No existe competencia real en el mercado colombiano. Oportunidad de definir la categoría.
3. **Recomendador inteligente**: No solo muestra beneficios, sino que recomienda activamente qué tarjeta usar por categoría de compra con un algoritmo de scoring ponderado.
