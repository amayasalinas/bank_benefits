# ELISEO — Maximiza tus beneficios

> La guía inteligente de beneficios de tarjetas de crédito y débito en Colombia.

## Sobre ELISEO

ELISEO es una aplicación web progresiva (PWA) que ayuda a los colombianos a aprovechar al máximo los beneficios de sus tarjetas bancarias: cashback, puntos, millas, acceso a salas VIP, descuentos y mucho más.

**Importante:** ELISEO no se conecta directamente con entidades financieras ni con franquicias vía API. La información es de carácter educativo e informativo.

## Funcionalidades

- **Mi Cartera** — Centraliza tus tarjetas de crédito y débito
- **Recomendador** — Descubre qué tarjeta usar según la categoría de gasto
- **Beneficios** — Información detallada de cashback, puntos, salas VIP, seguros y más
- **Ofertas** — Promociones y descuentos activos en Colombia
- **Perfil** — Gestión de cuenta y preferencias

## Bancos soportados

Bancolombia, Davivienda, BBVA Colombia, Banco de Bogotá, Banco Popular, Scotiabank Colpatria, Itaú Colombia, Banco de Occidente, AV Villas, Nu Colombia, Lulobank, RappiCard, Nequi.

## Stack tecnológico

- **Frontend:** React 18 + TypeScript + Vite
- **Estilos:** Tailwind CSS
- **Backend:** Supabase (PostgreSQL + Auth)
- **Deploy:** GitHub Pages
- **Hosting DB:** Supabase

## Desarrollo local

```bash
npm install
npm run dev
```

## Despliegue

El despliegue en GitHub Pages se hace automáticamente al hacer push a `main` via GitHub Actions.
